Setting up Debian jessie (8) on Mac Book Pro 11,5 (Retina mid 2015)

日本語¶

状況¶

一部の手順が不完全である。

制限¶

  • シャットダウン時に電源が落ちない
  • デフォルトのカーネル (3.16) では brcmfmac が無線 LAN をサポートしていない
  • より詳細な情報はこの 記事 を参照してください

大まかな手順¶

  1. jessie-backports の non-free, main, contrib レポジトリを追加

  2. 次のパッケージをインストールする

    ご用心

    この手順でカーネルが 4.7 にアップグレードされること

    • linux-image-4.7.0-0.bpo.1-amd64
    • firmware-brcm80211
    • firmware-linux
    • firmware-linux-nonfree

  3. タッチパッドの設定

    • Debian wiki の通り [1] ~/.Xmodmap の設定
    • Debian wiki の通り [2] /etc/X11/xorg.conf.d/synaptics.conf の設定

  4. バックライトの輝度を制御するため setpci -v -H1 -s 00:01.00 BRIDGE_CONTROL=0 を /etc/rc.local に追記する

  5. （任意）ファンクションキーの挙動を変更し、 Fn キーを押したときに特殊キーが押されるようにする [3]

    /etc/rc.local:

    echo 2 > /sys/module/hid_apple/parameters/fnmode

English¶

Status¶

There are still uncompleted steps.

Limitations¶

  • Unable to poweroff.
  • brcmfmac in default kernel (3.16) does not support Wi-Fi adapter.
  • Refere this article to know more details.

Rough steps¶

  1. Add a repository non-free, main, contrib of jessie-backports

  2. Install following packages

    ご用心

    kernel will be upgraded to 4.7 in this step.

    • linux-image-4.7.0-0.bpo.1-amd64
    • firmware-brcm80211
    • firmware-linux
    • firmware-linux-nonfree

  3. Configure touchpad

    • Configure ~/.Xmodmap as Debian wiki [1]
    • Configure /etc/X11/xorg.conf.d/synaptics.conf as Debian wiki [2]

  4. Add setpci -v -H1 -s 00:01.00 BRIDGE_CONTROL=0 to /etc/rc.local to control backlight brightness

  5. (Optional) Change function key behaviors [3]

    /etc/rc.local:

    echo 2 > /sys/module/hid_apple/parameters/fnmode
