2017-01-16 09:00:00

  • 無理せずに続けたいので、週一以上で書かないようにする

最近やってたこと¶

  • Runtastic Six pack [1] の Level 3 で限界を迎えた
    • 肩が弱いのでサイドプランクなどの肩を使うトレーニングで、 腹筋より先に肩が限界を迎えてしまう
    • ただ、肩の筋肉痛が来ず、筋肥大につながっている感もないので壁にぶつかっている状況
    • セルフウェイト (bodyweight) でどうにか肩を鍛えたい
    • 普通のクランチだとオールアウトまでが時間食ってツラいので、 Six pack のメニューでどうにかしたい
  • ビデオエイペックスさん [2] で MO ドライブを借りてデータサルベージを行った
    • キンコーズに MO ドライブあるという数年前の書き込みをネット上でちらほら見かけるが、 店頭で確認したところ、現在は全店舗で基本的に設置していないらしい
  • GAS でウェブアプリ組む方法 [4] について調べていた
    • One Drive の API トークンを取得する [5] ための雑アプリを組むために使う予定
  • フリーの DDNS と組み合わせてメールを受信できるサービスを探した
    • 最終的に Postmark [7] を使うことにした
    • Postmark は、受信だけであれば MX レコードだけで利用できる [8]
    • 最有力候補は mailgun [6] だったが、 mailgun は MX, TXT レコードの登録も必要で、 MX, TXT レコードを無料で追加できる DDNS サービスは見つけられなかった
    • DDNS には no-ip [9] を使うことにした
    • メール受信したいだけなので別にダイナミックでなくてもいいだけど。
  • 数理統計研究所の公開講座「 欠測データの統計科学 」に参加した
    • まとめて記事にする予定

  • Man page of bash(1) - Process Substitution

    • プロセス置換を使った際のリダイレクトの挙動がよくわからず、動作と man での説明を確認した

    • The name of this file is passed as an argument to the current command as the result of the expansion.

展開の結果、このファイル名（展開された FIFO や /dev/fd の名前）が現在のコマンドへ引数として渡される

    • >(list) や <(list) だけでリダイレクトされると考えていたが、誤解だった

    • プロセス置換だけではリダイレクトされず、 list への標準入出力へつながった FIFO や FD のパスへ置換される

    • 実例を見てみる

      $ # 例えば、次のようなコマンドを実行すると、 'hoge' と FD 名が出力される
$ echo hoge >(cat)
hoge /dev/fd/63
$ # このとき何が起きているかを -x オプションで表示してみると、
$ # プロセス置換の展開結果として FD 名が引数として渡されていることがわかる
$ bash -cx 'echo >(cat)'
+ echo /dev/fd/63
/dev/fd/63
++ cat

  • Handbook of Character Recognition and Document Image Analysis 3.1 Skew detection

    • 傾き角検出に projection profile analysis を使う例を探して見つけた書籍
    • この書籍の場合、各角度で水平方向の projection profile を取得し、分散が最大となる角度を水平としている

  • PHP: mysqli::poll - Manual

    • PHP の MySQLi は、非同期でクエリを投げられるらしく、実装を確認したかった
    • 実装 [11] を確認すると、非同期クエリが使えるのはビルド時に Mysqlnd が有効な場合になりそうだった
    • また、 libmysqlclient にはドキュメント化された API として非同期クエリの API が存在しないらしい [12]
    • したがって、 libmysqlclient の他言語バインディング、例えば MySQL-Python 、 を使う場合は、非同期クエリは使えないものと考えた方がよいのだろう
    • 備考として、 PHP の Mysqlnd には user handler plugin [13] というプラグインが存在しており、 Mysqlnd のより低レベルな API を PHP から利用することができる
